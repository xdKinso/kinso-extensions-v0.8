# Kinso Extensions

Paperback extensions for manga sources.

## Available Extensions

### Comix

Extension for reading manga from comix.to

### MangaPark

Extension for reading manga from mangapark.net

## Installation

Add this repository to Paperback:
https://xdkinso.github.io/KinsoExtension/

## Developer

Developed by Kinso - https://github.com/xdKinso

## Deployment

This repository automatically deploys to GitHub Pages when you push to the `main` branch.

### GitHub Pages Settings
Make sure GitHub Pages is configured in your repository settings:
1. Go to Settings → Pages
2. Source: GitHub Actions
3. The site will be published at: https://xdkinso.github.io/template-extensions/
